# Отчет по EDA и предобработке данных

## Ключевые выводы EDA

Данные содержат 2,452 финансовые операции с выраженным дисбалансом: расходы составляют 88.6% против 5.1% доходов. Категория "Food" доминирует с 36.95% всех операций, при этом наблюдается сильный дисбаланс между категориями - из 50 уникальных категорий многие представлены единичными наблюдениями. 

Анализ денежных потоков выявил значительную разницу в суммах: средний доход в 27 раз превышает средний расход, при этом 12.85% операций являются статистическими выбросами с суммами до 250,000 единиц. Качество данных требует улучшения из-за пропусков в столбцах Subcategory и Note, а также наличия опечаток в названиях категорий.

## Обоснование предобработки

Обработка пропущенных значений заполнением строкой 'UNKNOWN' была выбрана потому, что столбцы Subcategory и Note содержат значительные пропуски, и удаление соответствующих строк привело бы к потере существенной части данных. 

Преобразование даты в формат datetime с последующим извлечением признаков необходимо для лучшей подготовки данных в виле матрицы для обучения.

Удаление дубликатов критически важно для предотвращения смещения статистического анализа и исключения возможных ошибок двойного ввода данных, что в конечном счете улучшает качество данных для машинного обучения.

Векторизация текстовых полей с помощью CountVectorizer применяется к полю Note, поскольку текстовые описания содержат ценную семантическую информацию о характере транзакций. 

One-Hot Encoding для категориальных признаков был выбран потому, что в данных присутствуют 50 различных финансовых категорий типа "Food", "Transportation", "Investment", которые не имеют естественного порядка.
