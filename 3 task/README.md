# Analysis of bank transactions (3 задание Актионада 2025)

## Описание проекта

Проект посвящен анализу распределения банковских транзакций по различным категориям расходов. На основе датасета, содержащего 5389 транзакций, проводится исследование структуры потребительских расходов и выявление наименее представленных категорий.

## Основные результаты анализа

### Распределение транзакций по категориям

Данные демонстрируют сбалансированное распределение транзакций по 14 категориям расходов:

- **Общее количество транзакций**: 5389
- **Количество категорий**: 14
- **Диапазон транзакций на категорию**: от 347 до 417
- **Медианное значение**: 389.5 транзакций на категорию

### Ключевые показатели

- **Наиболее популярная категория**: Utilities (417 транзакций)
- **Наименее представленная категория**: Housing (347 транзакций)
- **5% наименее представленных категорий**: Housing (6.44% от общего объема)

### Визуализация

В проекте представлены следующие графики:
1. Столбчатая диаграмма распределения транзакций по категориям
2. Круговая диаграмма с долями каждой категории
3. Отдельная визуализация для наименее представленных категорий

## Структура данных

Исходный датасет содержит 20 столбцов, включая:
- Идентификаторы транзакций и счетов
- Даты и суммы транзакций
- Демографические данные клиентов
- Категории расходов
- Флаги мошенничества

## Заключение

Анализ показал, что данные хорошо сбалансированы и подходят для дальнейшего анализа и построения прогнозных моделей. Отсутствие сильного дисбаланса между категориями позволяет использовать стандартные методы машинного обучения без необходимости применения специальных техник обработки несбалансированных данных.
